◆ CanPlaceInSlot()

bool InventorySlot.CanPlaceInSlot ( ItemObject  _itemObject)

Checks if the item can be placed in the slot.

First it is checked if there is a restriction for the slot at all, if not return true. "AllowedItems.Length <= 0" all items are allowed, "_itemObject.data.Id < 0" the slot is empty. If only a specific item type is allowed, it is checked if the item type equal to the requested item type

Parameters
_itemObjectthe item to be placed in the slot
Returns
return true if the item is allowed to be placed in the slot. return false if not
