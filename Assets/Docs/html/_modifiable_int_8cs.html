delegate void ModifiedEvent ( )

A Delegate is a reference pointer to a method.

It allows us to treat method as a variable and pass method as a variable for a callback. When it get called, it notifies all methods that reference the delegate. Anyone can subscribe to the service and they will receive the update at the right time automatically.
