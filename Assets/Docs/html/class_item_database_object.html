Detailed Description

Hold all itemsObjects that exist in the game and gives them an ID.

This way you don't have to give each item an ID manually. A ItemDatabaseObject can be created in the editor. Extend ISerializationCallbackReceiver -> unity dont serialize dictionary. So we use the two callbacks to manually serialize it.
